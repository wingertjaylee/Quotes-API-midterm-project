## Overview
This project is a RESTful API built with PHP and PostgreSQL. It provides endpoints to manage quotes, authors, and categories.

## Tech Stack
- PHP
- PostgreSQL (Render)
- XAMPP
- GitHub

## API Endpoints

### Quotes
- GET /api/quotes
- GET /api/quotes/?id=1
- GET /api/quotes/?author_id=1
- GET /api/quotes/?category_id=1
- GET /api/quotes/?random=true

### Authors
- GET /api/authors
- GET /api/authors/?id=1

### Categories
- GET /api/categories
- GET /api/categories/?id=1

## Notes
- Database hosted on Render PostgreSQL
- API tested locally using XAMPP and Postman
